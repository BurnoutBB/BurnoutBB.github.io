Sposoby definowania zmiennych


Slowo kluczowe VAR

Zmienne zadeklarowane za pomoca var sa przypisywane do funkcji lub globalnego kontekstu, a ich zakres jest ograniczony do kontekstu, w ktorym zostaly zdefiniowane

var x = 7;


Slowo kluczowe LET

Zmienne deklarowane za pomoca let maja zasieg ograniczony do bloku, w ktorym zostaly zdefiniowane

let y = 18;


Slowo kluczowe CONST

Zmienne deklarowane za pomoca const sa stale i nie moga byc ponownie przypisane. Jednak wartosc obiektu, do ktorego odnosza sie, moze byc zmieniana

const z = 15;


Zmienne bez slowa kluczowego

Mozemy rowniez zadeklarowac zmienna poprzez przypisanie wartosci do niej bez uzycia slow kluczowych `var`, `let` lub `const` jednak taka zmienna bedzie globalna

zmienna1 = "wartosc";


Destrukturyzacja

Przypisanie wielu zmiennych jednoczesnie

var [a, b] = [1, 2];

let [c, d] = [3, 4];

const [e, f] = [5, 6];

+ dodawanie liczb oraz klejenie lańcuchow (konkatenacja)

- odejmowanie liczb

* mnozenie liczb

/ dzielenie liczb

++ inkrementacja ( zwiekszenie liczby o dokladnie 1 )

-- dekrementacja ( zmniejszenie liczby o dokladnie 1 )

% reszta z dzielenia ( tzw. modulo )

= przypisanie wartości do zmiennej

== porownanie wartości

< mniejsze od

<= mniejsze lub rowne od

> wieksze od

>= wieksze lub rowne od

! zaprzeczenie (negacja)

&& logiczne i (and)

|| logiczne lub (or)

+= skrocony zapis, np a += b odpowiada a = a + b

-= skrocony zapis, np a -= b odpowiada a = a - b

*= skrocony zapis, np a *= b odpowiada a = a * b

/= skrocony zapis, np a /= b odpowiada a = a / b

%= skrocony zapis, np a %= b odpowiada a = a % b




Okna dialogowe


ALERT Wyświetla prosty komunikat dla uzytkownika

alert("Tresc komunikatu");


CONFIRM Wyświetla komunikat z opcja potwierdzenia lub anulowania

var potwierdzenie = confirm("Czy na pewno chcesz kontynowac?");

if ( potwierdzenie ){

  // Kod ktory zostanie wykonany po potwierdzeniu

} else {

  // Kod ktory zostanie wykonany anulowaniu

}

var imie = prompt("Prosze wpisać swoje imie:")

if (imie != null) {

  // Kod ktory zostanie wykonany po wpisaniu imienia

} else {

  // Kod ktory zostanie wykonany po zostawieniu pola pustego

}

Pętle


Petla for

for (inicjalizacja; warunek; krok){

  // Kod do wykonania w każdej literacji

}

  • Inicjalizacja - Zazwyczaj jest to deklaracja i inicjalizacja zmiennej, która będzie używana jako licznik pętli. Na przykład 'let x = 0' przypisuje wartość zmiennej 'x' na zero

  • Warunek - Jeśli jest spełniony pętla kontynuuje swoje działanie, jeśli nie pętla kończy się.

  • Krok - Instrukcja wykonywwana w każdej literacji pętli, dla przykładu 'x++" będzie zwiększał wartość 'x' o 1 przy każdym wykonaniu pętli

Przykład użycia pętli for

for (let x = 0; x < 8; x++){

 document.write(x);

}

Tak zapisana pętla będzie dodawać do zmiennej 'x' 1 przy pomocy inkrementacji dopóki zmienna 'x' będzie mniejsza od 8

Petla while

while (warunek){

  // Kod do wykonania w każdej literacji

}

Pętla while wykonuje się do momentu kiedy warunek jest prawdziwy

Przykład użycia pętli while

let x = 0;

while (x < 10) {

  x += 3;

  document.write(x);

}

Tak zapisana pętla będzie dodawać 3 do zmiennej `x` tak długo aż jej wartość będzie mniejsza niż 10

Wynikiem takiej pętli będzie

369


Petla do... while

do {

  // Kod do wykonania w każdej literacji

} while (warunek)

Pętla do... while jest bardzo podobna do pętli while z tą różnicą że warunek jest sprawdzany dopiero po wykonaniu kodu a nie przed nim. Co oznacza że pętla zostanie wykonana co najmniej raz nawet jeśli warunek jest fałszywy na początku.

Przykład użycia pętli do...while

let x = 0

do {

 document.write(x);

  x++;

} while( x <= 12 );

Tak zapisana pętla będzie dodawać 1 do zmiennej `x` tak długo aż jej wartość będzie mniejsza lub równa 12

Zdarzenia


Zdarzenie onclick

To zdarzenie wystepuje gdy element zostanie kliknięty przez użytkownika

<input type="submit" onclick="funckja()" value="Naciśnij mnie" >

Tak zapisane zdarzenie onclick wywoła funkcję o nazwie "funkcja()" w momencie kiedy użytkownik naciśnie przycisk


Zdarzenie onmouseover

To zdarzenie występuje gdy element zostanie najechany myszką

<img src="obraz1.jpg" onmouseover="funckja2()" >

Tak zapisane zdarzenie onmouseover wywoła funkcję o nazwie "funckja2()" w momencie kiedy użytkownik najedzie myszką na zdjęcie


Zdarzenie onmouseout

To zdarzenie występuje gdy mysz zjedzie z elementu na który najechała, jest to odwrotność zdarzenia onmouseover

<img src="obraz1.jpg" onmouseout="funckja3()" >

Tak zapisane zdarzenie onmouseover wywoła funkcję o nazwie "funckja3()" w momencie kiedy użytkownik zjedzie myszką ze zdjęcia


Zdarzenie onchange

To zdarzenie występuje gdy wartość elementu zostanie zmieniona

<input type="number" onchange="funkcja4()">

Tak zapisane zdarzenie onchange wywoła funkcję "funkcja4()" za każdym razem kiedy wartość <input type="number" > ulegnie zmianie


Zdarzenie onfocus

To zdarzenie występuje gdy element dostaje fokus (Jest aktywny)

<input type="text" onfocus="funkcja5()">

Tak zapisane zdarzenie onfocus wywoła funckję "funckja5() w momencie gdy element <input type="text" > będzie aktywny


Zdarzenie onblur

To zdarzenie wystepuje gdy element traci fokus (Przestaje być aktywny)

<input type="text" onblur="funkcja6()">

Tak zapisane zdarzenie onblur wywoła funckję "funkcja6()" w momencie gdy element <input type="text" > przestanie być aktywny




Obiekty DOM


document.write

Ten obiekt jest wykorzystywany do generowania zawartości na stronie

document.write("Tekst który zostanie wyświetlony na stronie")

Jeśli chcemy generować kod języka HTML za pomocą obiektu document.write należy używać apostrofów zamiast cudzysłowów

Przykład użycia

document.write('<div id="mój_div"><p>Przykładowy tekst</p></div>);

Warto też tutaj wspomnieć o elemencie console.log który nie jest obiektem DOM ale ma podobne działanie do document.write z tą różnicą że generowana treść będzie w konsoli a nie na stronie

Przykład użycia

console.log("To jest komunikat dla konsoli")


document.getElementById

Tak zapisany obiekt pobierze wartość z elementu o id takim jak wpiszemy w nawias

var x = document.getElementById("pole_numeryczne").value;

W ten sposób możemy przypisać wartość z pola numerycznego o id="pole_numeryczne" do zmiennej `x`

Przykład użycia

function PobieranieDanych(){

  var x = document.getElementById("pole_numeryczne").value

  document.write("Pobrana wartość to: " + x)

}
