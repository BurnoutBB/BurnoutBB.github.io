Sposoby definowania zmiennych


Slowo kluczowe VAR

Zmienne zadeklarowane za pomoca var sa przypisywane do funkcji lub globalnego kontekstu, a ich zakres jest ograniczony do kontekstu, w ktorym zostaly zdefiniowane

var x = 7;


Slowo kluczowe LET

Zmienne deklarowane za pomoca let maja zasieg ograniczony do bloku, w ktorym zostaly zdefiniowane

let y = 18;


Slowo kluczowe CONST

Zmienne deklarowane za pomoca const sa stale i nie moga byc ponownie przypisane. Jednak wartosc obiektu, do ktorego odnosza sie, moze byc zmieniana

const z = 15;


Zmienne bez slowa kluczowego

Mozemy rowniez zadeklarowac zmienna poprzez przypisanie wartosci do niej bez uzycia slow kluczowych `var`, `let` lub `const` jednak taka zmienna bedzie globalna

zmienna1 = "wartosc";

let [c, d] = [3, 4];

const [e, f] = [5, 6];




Operatory i ich dzialanie

+ dodawanie liczb oraz klejenie lańcuchow (konkatenacja)

- odejmowanie liczb

* mnozenie liczb

/ dzielenie liczb

++ inkrementacja ( zwiekszenie liczby o dokladnie 1 )

-- dekrementacja ( zmniejszenie liczby o dokladnie 1 )

% reszta z dzielenia ( tzw. modulo )

= przypisanie wartości do zmiennej

== porownanie wartości

< mniejsze od

<= mniejsze lub rowne od

> wieksze od

>= wieksze lub rowne od

! zaprzeczenie (negacja)

&& logiczne i (and)

|| logiczne lub (or)

+= skrocony zapis, np a += b odpowiada a = a + b

-= skrocony zapis, np a -= b odpowiada a = a - b

*= skrocony zapis, np a *= b odpowiada a = a * b

/= skrocony zapis, np a /= b odpowiada a = a / b

%= skrocony zapis, np a %= b odpowiada a = a % b




Okna dialogowe


ALERT Wyświetla prosty komunikat dla uzytkownika

alert("Tresc komunikatu");


CONFIRM Wyświetla komunikat z opcja potwierdzenia lub anulowania

var potwierdzenie = confirm("Czy na pewno chcesz kontynowac?");

if ( potwierdzenie ){

  // Kod ktory zostanie wykonany po potwierdzeniu

} else {

  // Kod ktory zostanie wykonany anulowaniu

}


PROMPT Wyświetla okno z polem tekstowym, ktore umożliwia użytkownikowi wprowadzenie danych

var imie = prompt("Prosze wpisać swoje imie:")

if (imie != null) {

  // Kod ktory zostanie wykonany po wpisaniu imienia

} else {

  // Kod ktory zostanie wykonany po zostawieniu pola pustego

}

Pętle


Petla for

for (inicjalizacja; warunek; krok){

  // Kod do wykonania w każdej literacji

}

  • Inicjalizacja - Zazwyczaj jest to deklaracja i inicjalizacja zmiennej, która będzie używana jako licznik pętli. Na przykład 'let x = 0' przypisuje wartość zmiennej 'x' na zero

  • Warunek - Jeśli jest spełniony pętla kontynuuje swoje działanie, jeśli nie pętla kończy się.

  • Krok - Instrukcja wykonywwana w każdej literacji pętli, dla przykładu 'x++" będzie zwiększał wartość 'x' o 1 przy każdym wykonaniu pętli

Przykład użycia pętli for

for (let x = 0; x < 8; x++){

 document.write(x);

}

Tak zapisana pętla będzie dodawać do zmiennej 'x' 1 przy pomocy inkrementacji dopóki zmienna 'x' będzie mniejsza od 8

Petla while

while (warunek){

  // Kod do wykonania w każdej literacji

}

Pętla while wykonuje się do momentu kiedy warunek jest prawdziwy

Przykład użycia pętli while

let x = 0;

while (x < 10) {

  x += 3;

  document.write(x);

}

Tak zapisana pętla będzie dodawać 3 do zmiennej `x` tak długo aż jej wartość będzie mniejsza niż 10
