Utworzenie bazy danych

create DATABASE



Usuwanie bazy danych

DROP DATABASE nazwa_bazy



Dodanie tabeli do bazy danych

CREATE TABLE nazwa_tabeli(nazwa_kolumny typ_danych DODATKOWE_ARGUMENTY)

Podstawowe typy danych:

Dodatkowe argumenty np:

  AUTO INCREMENT

  PRIMARY KEY

  FOREGIN KEY



Dodawanie danych do tabeli:

INSERT into nazwa_nazwa tabeli VALUES(wartosci, oddzielone, przecinkami)

W przypadku wpisywania danych tekstowych musimy uzyc cudzysłowow lub apostrofow

INSERT into nazwa_tabeli VALUES("Wartosci","tekstowe")

W przypadku liczby zmiennoprzecinkowej warto pamietac aby uzyc kropki a nie przecinka



Dodawanie kolumn do tabeli:

ALTER TABLE nazwa_tabeli ADD(nazwa_kolumny typ_danych)

Aby dodac kolumne w konkretnym miejscu nalezy uzyc AFTER

ALTER TABLE nazwa_tabeli ADD nazwa_kolumny typ_danych AFTER nazwa_innej_kolumny



Usuwanie kolumn:

ALTER TABLE nazwa_tabeli DROP nazwa_kolumny



Zmiana typu danych w kolumnie:

ALTER TABLE nazwa_tabeli MODIFY nazwa_kolumny typ_danych



Zmiana nazwy kolumny:

ALTER TABLE nazwa_tabeli CHANGE stara_nazwa nowa_nazwa typ_danych



Zmiana nazwy tabeli:

ALTER TABLE nazwa_tabeli RENAME nowa_nazwa



Wyszukiwanie danych w tabeli:

SELECT szukana_wartosc FROM `nazwa_tabeli` WHERE inna_wartosc=1

Dla przykladu wyszukanie emaila uzytkownika o ID=1 z tabeli uzytkownicy:

SELECT email FROM `uzytkownicy` where user_id=1

Warto pamietac ze nie uzywamy tutaj ani cudzyslowiu ani apostrofu a GRAWISU `
